Windows Server 2019 on VirtualBox

Detailed Documentation About This Project Below

Downloads and VirtualBox Setup

The first step is to download the Windows Server 2019 .ISO from Microsoft's evaluation website along with VirtualBox from Oracle's website.

In this step, I created a new virtual machine in VirtualBox with the "New" button. I then named the VM "Windows Server 2019" to specify what it is. The type of VM is Windows, and the version is Windows 2019 (64-bit).

After creating the VM I will change the network adapter type from NAT to Bridged Adapter. This gives the VM it's own IP address on the host computer's network instead of using an internal network inside of Virtual Box. This will help when connecting other VMs to the Domain.
