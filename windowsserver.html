Windows Server 2019 on VirtualBox

Detailed Documentation About This Project Below

The first step is to download the Windows Server 2019 .ISO from Microsoft's evaluation website along with VirtualBox from Oracle's website.

In this step I created a new virtual machine in VirtualBox with the "New" button. I then names the virtual machine Windows Server 2019 to tell what it is. The type of virtual machine is windows and the version is Windows 2019 (64-bit) respectively.
